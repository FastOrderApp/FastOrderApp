# 🐧 학교 앞 포장/픽업 주문 앱, 패패오더

## 📖 프로젝트 소개
- 패패오더는 미리 주문하고 예약하는 기능으로 공강시간을 지키고싶은 대학생과 사장님을 연결해주는 주문 앱입니다.
- 지도에서 주문 가능한 가게를 음식 카테고리 별로 볼 수 있습니다.
- 자주 주문하는 메뉴를 메인 화면에서 원터치로 주문이 가능합니다.
- 사장님용 포스 앱을 통해 사장님이 주문 정보를 빠르게 받아볼 수 있습니다.
